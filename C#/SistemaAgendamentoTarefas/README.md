## Sistema de Agendamento de Tarefas

  Neste desafio foi proposto construir um sistema de agendamento de tarefas no formato de CRUD, ou seja, em que seja possível **criar**, **visualizar**, **atualizar** e **deletar** uma tarefa.
  
  O código base para o desafio se encontra no [repositório](https://github.com/digitalinnovationone/trilha-net-api-desafio) da Digital Innovation One.
